# 📝 Nuestro Blog Privado

Un blog privado y especial creado solo para ti y tu novia. Fácil de usar y completamente gratuito en GitHub Pages.

## ✨ Características

- ✅ **Privado**: Solo ustedes dos pueden publicar (protegido con contraseña)
- ✅ **Fácil de usar**: Interfaz simple e intuitiva
- ✅ **Compartido**: Ambos pueden publicar desde cualquier lugar
- ✅ **Gratuito**: Funciona perfectamente en GitHub Pages
- ✅ **Sin base de datos**: Todo se guarda en el navegador
- ✅ **Diseño romántico**: Especialmente pensado para ustedes

## 🚀 Cómo configurar

### 1. Crear los archivos
Sube estos tres archivos a tu repositorio de GitHub Pages:
- `index.html` (la página principal del blog)
- `admin.html` (el panel de administración)
- `README.md` (estas instrucciones)

### 2. Configurar la contraseña
En el archivo `admin.html`, busca esta línea:
